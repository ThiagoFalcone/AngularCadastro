Sobre o Projeto

Esta aplicação foi desenvolvida para cadastrar alunos de forma simples utilizando Angular.

Ela demonstra o uso de:

  • Componentes independentes (standalone)
  • Rotas com navegação entre páginas
  • Formulários com ngModel (two-way binding)
  • Diretivas estruturais como *ngIf e *ngFor
